                    📖 사전 관리란? 검색 품질을 높이기 위해 동의어·불용어·사용자사전을 등록합니다.

                    • synonym (동의어): 같은 의미의 단어를 매핑합니다. 예) AI → 인공지능

                    • stopword (불용어): 검색에서 무시할 조사/접속사를 등록합니다. 예) 에, 은, 는

                    • user (사용자사전): 오타를 올바른 단어로 교정합니다. 예) 계휙서 → 계획서

                    📁 SMB 색인이란? 네트워크 공유 폴더(SMB/CIFS)에 있는 파일을 자동으로 수집·색인합니다.

                    • name: 소스를 구분할 이름 (예: 본사공유, R&D자료실)

                    • share_path: SMB 경로 (예: \\\\server\\share\\docs)

                    • username/password: 인증이 필요한 경우 입력
                

                

                    
                    
                    
                    
                    
                

                

                    
                    
                    
                    
                

                

                    
ID이름경로활성최근 동기화관리

                

                

            


            
            

                

                    🗄️ DB 색인이란? 외부 데이터베이스에 저장된 텍스트 데이터를 SQL 쿼리로 추출하여 OpenSearch에 색인합니다.

                    • db_type: mysql, oracle, postgresql, sqlite 중 선택

                    • connection_url: DB 접속 문자열 (예: mysql+pymysql://user:pass@host/db)

                    • query_text: 색인할 데이터를 추출하는 SELECT 쿼리

                    • chunk_size: 텍스트를 분할하는 단위 (기본 500자)

                    ⏰ 자동 색인 스케줄러란? 설정한 주기(분)마다 SMB·DB 소스를 자동으로 동기화하여 최신 상태를 유지합니다.

                    • 간격(분): 동기화 주기 (최소 10분, 기본 120분)

                    • 시작 후 백그라운드에서 자동 실행되며, 별도 중지 전까지 계속됩니다.
                

                

                    
                    
                    
                    
                    
                    
                

                

                    
항목값

                

                

            


            
            

                

                    🔐 볼륨/SSL 관리란? OpenSearch 인덱스(볼륨) 생성 및 HTTPS 인증서의 만료 상태를 관리합니다.

                    • 볼륨 생성: 커스텀 OpenSearch 인덱스를 생성합니다.

                    • 인증서 상태: SSL 인증서의 만료일을 모니터링합니다.

                    • 갱신 스크립트: Let's Encrypt 또는 자체 서명 인증서를 갱신합니다.
                

                

                    
                    
                    
                    
                

                

                    
                    
                    
                    
                    
                

                

                    
파일명발급대상만료일남은일수상태

                

                

            


            
            

                

                    📊 검색 품질 관리란? 검색 결과의 순위(Scoring)를 결정하는 가중치를 실시간으로 조정하고, 검색 품질을 평가합니다.

                    • 제목 구문: 검색어가 제목에 정확히 포함될 때 부여하는 점수 (기본 20)

                    • 제목 AND: 검색어 단어가 제목에 모두 존재할 때 점수 (기본 10)

                    • 본문 구문: 검색어가 본문에 정확히 포함될 때 점수 (기본 5)

                    • 본문 AND: 검색어 단어가 본문에 모두 존재할 때 점수 (기본 1)

                    • AI 벡터: 의미 유사도 기반 벡터 검색 가중치 (기본 1.0)

                    • 초성: 초성 검색 매칭 가중치 (기본 10)

                    ⚠️ 데이터 초기화 — OpenSearch 인덱스와 DB의 데이터를 모두 삭제합니다.

                    • [전체 데이터 초기화]: 문서 + 검색로그 + 최근검색어를 모두 삭제합니다.
