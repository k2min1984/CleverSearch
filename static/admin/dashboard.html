운영 대시보드 안내

최근 검색/실패/소스/인증서 상태를 요약해 보여줍니다.

• 조회일수 입력칸: 최근 N일(기본 7일) 기준으로 통계 범위를 정합니다. 값을 바꾸면 대시보드 지표와 차트가 해당 기간으로 다시 조회됩니다.

• 스모크 실행: 클릭 즉시 시스템 점검을 1회 실행하고, 하단 출력창에 결과 JSON이 표시됩니다. `status=success`면 통과, `status=fail`이면 `stderr`에 실패 원인이 표시됩니다. (admin 권한 필요)

• 디버그: 클릭할 때마다 하단 디버그 패널(health/summary/trend 원본 JSON)의 표시/숨김이 전환됩니다. 데이터 자체를 변경하지는 않습니다.
