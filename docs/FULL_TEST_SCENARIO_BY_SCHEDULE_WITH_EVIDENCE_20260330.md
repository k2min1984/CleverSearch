## 1. 실행 규칙

- 각 일정번호를 테스트한 뒤 완료 체크를 한다.
- 각 일정번호마다 최소 2장 캡처를 남긴다.
- 캡처 파일명은 아래 규칙을 따른다.

캡처 파일명 규칙:

- SCH{번호}\_STEP_01.png
- SCH{번호}\_STEP_02.png
- 예시: SCH24_STEP_01.png, SCH24_STEP_02.png

권장 저장 경로:

- docs/evidence/20260330/

## 2. 공통 증적 기준

- 실행 화면 캡처: 실제 클릭/입력/실행 화면
- 결과 화면 캡처: 성공 메시지/결과 목록/상태 변화
- API 항목은 응답 JSON 화면 또는 터미널 출력 캡처
