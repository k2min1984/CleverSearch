## 2. DB 관련 변경

### 2.1 멀티 DB 타입 지원

**변경 파일**: `app/core/config.py`, `app/core/database.py`, `alembic/env.py`, `.env.example`, `requirements.txt`

기존 PostgreSQL 고정 구조에서 Oracle, MariaDB, MySQL, PostgreSQL 4종 DB를 지원하도록 변경.

| 설정 항목 | 설명 |
|-----------|------|
| `DB_TYPE` | `postgres` / `mysql` / `mariadb` / `oracle` |
| `DB_HOST`, `DB_PORT`, `DB_NAME` | 접속 정보 |
| `DB_USER`, `DB_PASSWORD` | 인증 정보 |
| `DB_ORACLE_USE_SERVICE_NAME` | Oracle SID/서비스명 분기 |

- `config.py`: `_build_database_url()` classmethod 추가, `quote_plus()`로 특수문자 안전 처리
- `database.py`: DB 타입별 엔진 생성 분기 (sqlite/oracle/mysql-mariadb/postgres)
- `requirements.txt`: `pymysql`, `oracledb` 드라이버 추가

### 2.2 자격증명 AES-256-GCM 암호화

**변경 파일**: `app/utils/crypto.py` (신규), `app/services/system_service.py`, `app/core/config.py`, `.env.example`

DB 연결 URL, SMB 비밀번호 등 민감 정보를 평문 저장하지 않도록 암호화 적용.

- `crypto.py`: `encrypt()` → `"enc::Base64(...)"` 형식, `decrypt()` → 평문 복원
- `CREDENTIAL_SECRET` 환경변수를 SHA-256 해싱하여 AES 키로 사용
- 하위 호환: `enc::` 접두사 없으면 평문 그대로 반환
- `system_service.py`: SMB 비밀번호 / DbSource 연결 URL 저장 시 암호화, 사용 시 복호화

### 2.3 DB 접속 오류 수정

**증상**: `'utf-8' codec can't decode byte 0xb8` 에러 반복 발생

**원인**: 앱이 로컬 PostgreSQL(포트 5432, 한국어 로캘)에 접속하고 있었음. 실제 Docker 컨테이너(`postgres_pgvector`)는 포트 55432에서 동작 중.

**수정 내용**:
- `.env`: `DB_PORT=55432`, `DB_USER=cleversearch`, `DB_PASSWORD=dudflgks12!@`, `DB_NAME=cleversearch_db`
- `config.py`: `_build_database_url()`에서 `quote_plus()`로 `!@` 특수문자 이스케이프 처리

## 3. 파일 파서 변경

### 3.1 변경 파일 목록

| 파일 | 변경 내용 |
|------|-----------|
| `app/core/file/image.py` | PaddleOCR 전환, Tesseract 폴백 유지 |
| `app/core/file/excel.py` | `extract_text()` str 반환, XLS xlrd 폴백 추가 |
| `app/core/file/office.py` | 테이블/노트/헤더/푸터 추출, `extract_text()` 통합 진입점 추가 |
| `app/core/file/pdf.py` | pdfplumber 통합, fitz → PaddleOCR 폴백 (스캔 PDF 대응) |
| `app/core/file/hwp.py` | 제어문자 정리 내장, `extract_text()` 추가, bare except 제거 |
| `app/core/file/__init__.py` | 중복 import / `__all__` 제거 |
| `app/services/indexing_service.py` | 인라인 로직 제거, 모듈 위임 호출로 통일 |
| `requirements.txt` | paddlepaddle, paddleocr, opencv-python-headless 추가 |

### 3.2 image.py — OCR 엔진 전환

**Before**: Tesseract 단독 (치명적 오타 `iㄴmage_to_string` 포함)

**After**:
- PaddleOCR (한국어, CPU, 신뢰도 0.6+) → Tesseract 폴백 2단계 구조
- 싱글턴 패턴 (`_get_paddle_ocr()`) — 모델 1회만 로드
- PaddleOCR 미설치 시 자동으로 Tesseract만 사용

### 3.3 excel.py — str 반환 및 XLS 지원

- `extract_text(content: bytes) -> str` 추가
- XLSX(openpyxl) 우선 → XLS(xlrd) 폴백
- 에러 시 빈 문자열 반환
- `parse_excel_all_sheets()` 레거시 호환 유지

### 3.4 office.py — 추출 범위 확대 및 통합

- **Word**: 문단 + 테이블 + 헤더/푸터 추출, `[[Page 1]]` 마커 내장
- **PPT**: 셰이프 텍스트 + 테이블 + 슬라이드 노트, 슬라이드별 `[[Page N]]` 마커
- `extract_text(content, ext)` 통합 진입점 추가 (docx/pptx 자동 분기)

### 3.5 pdf.py — pdfplumber 통합

- 1차: pdfplumber 텍스트 추출 (페이지별 `[[Page N]]` 마커)
- 2차: 텍스트 없으면 PyMuPDF(fitz) → PaddleOCR 폴백 (스캔 PDF)

### 3.6 hwp.py — 안전성 강화

- `_extract_text()` 내부 제어문자 정리 + `[[Page 1]]` 마커 생성
- `extract_text()` 외부 호출용 래퍼 추가
- `except:` → `except Exception:`, `print()` 디버그 제거, 미사용 `_clean_text()` 제거

### 3.7 indexing_service.py — 호출 구조 통일

**Before**: 각 포맷별 10~20줄 인라인 로직, `image._extract_text()` 호출 → AttributeError

**After**:
```python
if ext in ["hwp", "hwpx"]:
    text_content = hwp.extract_text(content, ext)
elif ext == "pdf":
    text_content = pdf.extract_text(content)
elif ext in ["docx", "pptx"]:
    text_content = office.extract_text(content, ext)
elif ext in ["xlsx", "xls"]:
    text_content = excel.extract_text(content)
elif ext in ["jpg", "jpeg", "png"]:
    text_content = image.extract_text(content)
elif ext == "txt":
    text_content = content.decode("utf-8", errors="ignore")
```

- 불필요 import 제거 (`io`, `re`, `pdfplumber`, `pandas`, `docx`, `pptx`)
- `sanitize_text()` 후 중복 `re.sub` 제어문자 제거 라인 삭제

## 5. 호출 흐름도

```
POST /upload
  └─ IndexingService.index_bytes()
       └─ _extract_text(filename, content)
            ├─ hwp.extract_text(content, ext)     → HWP/HWPX 파싱
            ├─ pdf.extract_text(content)           → pdfplumber → PaddleOCR 폴백
            ├─ office.extract_text(content, ext)   → DOCX/PPTX (테이블/노트 포함)
            ├─ excel.extract_text(content)         → XLSX/XLS (xlrd 폴백)
            ├─ image.extract_text(content)         → PaddleOCR → Tesseract 폴백
            └─ content.decode("utf-8")             → TXT
       └─ sanitize_text() → 중복체크 → 임베딩 → OpenSearch 색인
```
